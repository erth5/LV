## Prepare Frontend

install dependencies
````shell
npm install
````
Run all Mix tasks
````shell
npm run dev
````
Run all Mix tasks and minify output
````shell
npm run prod
````
Run and check (in local env some Outliers)
````shell
npm run watch
````
force all files
````shell
npm run watch-poll
````
recommend css framework
````shell
npm install -D tailwindcss
npx tailwindcss init
````
